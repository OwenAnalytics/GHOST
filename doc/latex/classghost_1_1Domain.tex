\subsection{Detailed Description}
This class encodes domains of your C\+S\+P/\+C\+OP variables. You cannot inherits your own class from \hyperlink{classghost_1_1Domain}{Domain}. 

\hyperlink{classghost_1_1Domain}{Domain} is the class implementing variable domains, ie, the set of possible values a variable can take. In G\+H\+O\+ST, such values must be integers, possibly positive, negative or both.
