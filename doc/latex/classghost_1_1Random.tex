\subsection{Detailed Description}
\hyperlink{classghost_1_1Random}{Random} is the class coding pseudo-\/random generators used in G\-H\-O\-S\-T. 

\hyperlink{classghost_1_1Random}{Random} use the C++11 Mersenne Twister (mt19937) as a pseudo-\/random generator.

Seeds are generated by C++11 random\-\_\-device.
